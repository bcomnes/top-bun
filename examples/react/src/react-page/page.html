React with TypeScript in DOMStack

This example demonstrates client-side rendering with React and TypeScript in DOMStack.

How It Works

This static HTML page includes a mount point that React will use to render interactive components.

The client.tsx file contains React components with TypeScript that will be mounted to the container below.

Implementation Details

Unlike Preact examples that support isomorphic rendering, this example:

  • Runs exclusively in the browser
  • Uses React instead of Preact
  • Mounts to an empty container after page load
  • Uses TypeScript and TSX with custom ESBuild configuration
  • Provides type safety with TypeScript interfaces and type annotations
